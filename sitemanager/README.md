A Rust library for interacting with the UniFi Site Manager API.

## Features

### Implemented
- [x] List Sites
- [x] List Hosts
- [x] List SD-WAN Configs
- [x] Get ISP Metrics

### Not Yet Implemented
- [ ] Get Host by ID
- [ ] List Devices
- [ ] Query ISP Metrics
- [ ] Get SD-WAN Config by ID
- [ ] Get SD-WAN Config Status

## Usage

Add this crate to your `Cargo.toml`:

```toml
[dependencies]
sitemanager = "0.1.0"
```
